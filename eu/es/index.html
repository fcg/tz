西班牙买房移民

西班牙概况

西班牙位于西南欧，是欧洲第三大国家，首都是马德里。主要城市为马德里、巴塞罗那、马拉加。语言有西班牙、加泰罗尼亚语、巴斯克语、加利西亚语。

买房项目介绍

2013年9月27日，西班牙颁布了新的移民法案，为投资移民敞开了更广阔的大门；2015年7月31日，法案更新，实现了三代移民。

具体规则：

  • 投资50万欧元或以上价格的房产
  • 可续签居留许可：1+2+5+5...

房产类型可以是：民用住宅、商业用房、不限制土地类型。

房产数量： 一套或多套总价位50万欧元；如果是多个家庭合买，则需要平均每个家庭50万欧元。

可续签居留许可

可续签居留许可申请条件

主申请人：

  • 年满18周岁；
  • 无犯罪记录；
  • 购买健康保险；
  • 经济来源证明。

附属申请人：

  • 主申请人合法配偶、18周岁以下子女、年满18周岁未婚且经济不独立的子女；
  • 主申请人和配偶的经济不独立的父母；
  • 无犯罪记录；
  • 购买健康保险。

可续签居留许可申请流程

  1. 申请商务考察签证，登录西班牙选购房产；
  2. 交付房产定金，与律师签订授权协议，获取合同，开立银行账户；
  3. 回国
    1. 国内转账房款和税费到西班牙账户，授权律师转账给开发商；
    2. 获取地契，房产注册证明和赋税证明；
    3. 在中国递交居留申请，20个工作日审批通过后获得一年有效期签证。
  4. 回国
    1. 国内转账房款和税费到西班牙账户
    2. 在中国递交居留申请，获得6个月有效期的签证。注：如果6个月未获得地契，可以申请下一个6个月签证。
    3. 获取地契，房产注册证明和赋税证明。登录西班牙换取居留卡。

可续签居留许可续签流程

  1. 有效期内登录西班牙一次；
  2. 房产持有证明+赋税证明；
  3. 递交申请，录入指纹；
  4. 领取居留卡。

可续签居留许可项目优势

  • 三代移民，子女父母一同拿身份；
  • 子女续签无限制，一朝获签终身有效；
  • 可工作可经商；
  • 操作周期短，6个月拿身份；
  • 无需商业学术背景；
  • 自由出行26个申根国；
  • 高回报，升值空间大；
  • 无语言要求；
  • 无居住要求；
  • 免费公立教育、医疗。

非盈利居留许可

非盈利居留许可介绍

非盈利居留许可法案：

  • 建议购买30万欧元或以上价格的房产；
  • 3m+1+2+2... 非盈利居留许可。

2000年国家组织法规定，给予外国人融入西班牙社会的权利和自由。

2011年国家公告再次修订细节，鼓励外国人来西班牙消费。

非盈利居留许可申请条件

主申请人：

  • 年满18周岁；
  • 无非法滞留西班牙记录；
  • 在过去5年内居住过的国家无犯罪记录；
  • 有足够的生活来源；
  • 有公共或者私人医疗保险；
  • 没有患对公共健康有严重影响的疾病。

附属申请人：

  • 主申请人合法配偶、18周岁以下子女、年满18周岁的残疾子女；
  • 无犯罪记录；
  • 充足资金证明；
  • 购买健康保险；
  • 体检报告。

非盈利居留许可申请流程

  1. 去西班牙驻华领事馆递交申请及所需文件；
  2. 审核评估（一个月）；
  3. 通过后可以领取3个月有效期的申根签证；
  4. 登录西班牙，换取居留卡。
  5. 如果未通过，可以在2个月后再次申请，如果第二次仍被拒签，则永久封档。

非盈利居留许可续签流程

  1. 持有有效的临时居留许可；
  2. 经济能力证明及无从事盈利性工作证明；
  3. 有公共或私人医疗保险；
  4. 支付办理手续费票据；
  5. 未成年子女需提供在读证明；
  6. 积极融入社会证明，语言证明（可以加分）

非盈利居留许可优缺点

  • 申请门槛低；
  • 自由出行26个申根国；
  • 子女边读书边拿身份；
  • 无获批保证；
  • 无投资回报；
  • 融入社会难；
  • 续签无保障。

永居国民身份

永居： 5年内居住达到4年零2个月，且每年至少183天。

入籍： 通过入籍考试，且在西班牙连续居住10年。

新生儿满足下列条件可以直接入籍：

  • 在西班牙出生；
  • 在西班牙居住超过1年；
  • 父母至少一方有合法居留权。
