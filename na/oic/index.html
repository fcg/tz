OINP商业移民 - 公司类别

OINP商业移民公司类别适用于希望将现有国际公司业务拓展到安省或在安省购买现有企业的申请人。一旦企业成功建立并履行了履约协议的义务，最多有五名在企业建立、持续运营以及企业成功中发挥重要作用的员工可通过安省省提名获得永居身份。

申请条件 - 飞出国

  1. 公司要求：
    • 提交申请之时，公司必须已经建立了至少36个月；
    • 合格的公司结构包括：公营公司、独资企业和合伙企业；
  2. 投资要求：
    • 公司必须至少投资500万加币用于在安省拓展业务或者购买现有企业；
    • 投资资金必须为合法所得；
    • 拟议投资活动必须能对安大略省产生重大经济效益，这将通过对公司商业计划的评估来确定；
  3. 当地创造就业要求：
    • 公司每为一名关键员工申请移民，则需要为安省的加拿大公民或永居居民创造5个新的长期全职工作岗位，例如，提名5名关键员工，则需要在企业中为安省的加拿大公民或永居居民创造25个长期全职工作岗位；
    • 薪资必须达到或高于该职位的现行工资水平；
    • 长期全职工作是指12个月内至少持续工作1560小时的有偿工作（不包括季节性或兼职工作、分包商或代理、在家办公或远程办公）；
    • 在提交最终报告前，该岗位必须被持续填补至少10个月；
  4. 一般要求：
    • 母公司和在安省拟议的新企业之间必须有明确的结构联系（如附属公司、分支机构或子公司）；
    • 拟议企业必须打算通过出售商品和/或服务获得利润；
    • 基本收入来源必须来自积极（赚取）收入，而不是被动（非盈利）收入；
    • 拟议企业必须符合加拿大的法律要求以及所有监管行业和许可证要求；
      • 如购买现存企业，该企业必须是在安省成立的；
    • 拟议企业必须符合安省的所有劳动法，包括但不限于就业标准、健康和安全以及劳资关系立法；
    • 拟议企业必须是在安省的常设企业，不可以是项目型/季节性企业；
    • 第三方机构必须是附表一和附表二中的银行或机构投资者；
    • 根据IRPR 87（5）（6）段落的规定，拟议企业不得参与与移民挂钩的投资计划；
    • 必须提交商业计划书；
  5. 关键员工及职位要求：
    • 第一阶段：关键员工申请要求：
      • 关键员工申请人必须对拟议企业的建立和运营至关重要；
      • 关键员工申请人必须具备公司现有业务的高级、行政、管理或专业知识能力（职业属于NOC 0和A类）；
      • 只有一名关键员工可提名NOC A类职业，其他员工必须是提名NOC 0类职业；
      • 关键员工申请人在过去60个月内必须有至少36个月的公司预期工作经验（将在安省从事的工作职位），关键员工申请人在提交申请前12个月必须在该职位上为公司持续工作；
      • 关键员工申请人不得与公司执行董事会或董事会的任何成员或在公司持股超过10%的股东有直系亲属关系，直系亲属包括配偶、父母、兄弟姐妹、儿女、祖父母及配偶的直系亲属；
      • 关键员工申请人不得在母公司中持有股份，除非该股权是作为员工合同中一部分的薪酬待遇而获得的，在这种情况下，关键员工持股不得超过公司业务的10%,证明股权为薪酬待遇一部分的支持文件必须包含在概述公平条款的申请书中（例如就业协议）；
      • 安省拟议企业中关键员工的职位必须是长期全职的，并且需符合安省特定就业地区的工资中位数水平。
    • 第二阶段：关键员工提名要求：
      • 职位要求：关键员工申请人必须从事OINP批准的工作职位，并执行批准的工作职责，薪资水平达到安省现行工资标准；
      • 居住要求：关键员工申请人每年必须在安省实地居住至少9个月，持临时工作许可来支持企业经营；
      • 语言要求：关键员工申请人必须具备CLB 4或更高的语言水平；
  6. 购买安省现有企业的要求：
    • 购买之前60个月内，该现有企业必须由同一所有者持续经营（要求提供所有权证明以及购买意向书或出售协议）；
    • 该企业必须在安省注册成立；
    • 购买必须是将以前所有者100%的所有权转让给公司，这意味着之前的所有者不能保留任何公司股份；
    • 企业不得是OINP商业移民提名人之前拥有或当前经营的，必须保留所有现有长期全职工作岗位，并且为每个关键员工申请人额外创建5个长期全职工作岗位，必须至少保持现有员工的工资水平和工作条件；
    • 拟议商业计划书必须表明有扩大业务的打算；
    • OINP不鼓励申请人在签署履约协议以及从IRCC获得有效短期工作许可前进行任何投资，但可以接受提交申请前12个月进行的投资，在申请获批前签署协议购买企业和/或进行存款需自负风险，如已在该省投资，OINP不保证批准申请；
  7. 土地开发及租赁公司附加要求：
    • 每年最多3个项目；
    • 公司至少投资1000万加币；
    • 拟议活动必须是长期、持续进行的经营（非项目型），目的是扩大或改善土地现状，并确保投资公司在拟议业务中发挥持续积极的管理作用；
    • 合格的租赁公司必须生产出拟议的商品或提供服务，而不仅仅依靠管理其他公司的经营而获利；
    • 合格的土地开发公司必须证明拟议的业务是长期和非投机性的，相关土地开发将成为核心企业的组成部分；
    • 拟议的投资金额不能仅用于购买土地和建筑成本；
    • 必须提供当地现场认证文件和/或相关的当地规划机构批准文件；
    • 在监督期内，公司及关键员工申请人必须证明履约协议中所列出的承诺被满足，用于设立拟议公司以及支持持续经营，以便有资格获得提名。

申请流程 - 飞出国

第一阶段：

  1. 准备：
    • 公司和关键员工以及拟议业务所有各方面必须符合该移民类别的要求，然后再提交在线申请；
  2. 如何在线申请：
    • 使用OINP e-Filing Portal学习如何在线申请；
  3. 完成在线申请：
    • 提交完整的在线申请，必须确保：
      • 回答在线申请中的所有必答栏目；
      • 上传所有需要的支持文件，包括五个关键员工的关键员工申请表；
      • 阅读并同意所有声明和授权；
      • 使用电子支付为所有关键员工支付申请费（每人3500加币）；
  4. 申请后：
    • 在线申请中提供的所有信息都将得到验证，以确保其准确性、合法性和完整性；
    • 只有经过OINP审核完整性才可将申请视为完整的，不完整的申请不会被受理，并且会被退还申请费；
    • 作为申请评估的一部分，申请人将被要求参加强制性面试；
  5. 履约协议：
    • 如果申请获批，申请人需要签署一份履约协议，该协议概述了申请人在安省投资以及创造就业的承诺。

第二阶段：

  1. 临时工作许可（关键员工到达前）：
    • OINP将向关键员工申请人发出临时工作许可支持函，该支持函可用来向IRCC申请临时工作许可，以便申请人在安省创建企业（注意IRCC保留发出临时工作许可的权利）；
  2. 企业建立（关键员工到达后）：
    • 一旦所有关键员工申请人到达安省，将有20个月的时间来创建企业以及实施商业计划；
  3. 提名：
    • 如果履约协议中的承诺被实现，关键员工申请人就可以有资格通过OINP提名申请永居；
  4. 提名后：
    • OINP成功提名后六个月内，必须向IRCC提交永居签证申请，提名后对企业的监督将会持续36个月，如果所创建的公司或关键员工申请人不再履行履约协议中的承诺，OINP有权撤销提名。

申请费用 - 飞出国

每个关键员工3500加币，只能用信用卡（VISA或MasterCard)电子支付。

A. 公司文件
1.商业计划书 商业计划书副本，包括但不限于以下内容：财务和税务报表、销售预测、竞争对手分析、行业统计、市场趋势数据、雇员配置及人力资源计划、风险评估、执照或许可申请、行业协会会员资格。
1.1 财务分析 以下文件副本：收入报表：四年期预计收入报表（包括对所有收入和支出的明确假设）；资产负债表：四年期预计资产负债表（包括对所有资产和负债的明确假设）；现金流动报表：四年期预计现金流动报表（包括明确的假设）。
2.组织结构图 表明关键员工在公司作用的组织结构图副本
3.税务登记 公司税务登记证副本
4.股权协议 证明公司股东关系的股权协议副本
5.公司和法律结构文件 证明当前公司和法律结构的文件副本，包括授权签署官员、股东、股本和经营历史；
6.财务报表和商业银行报表 公司所有年份（过去60个月内至少36个月的）的财务报表和商业银行报表副本
7.公司章程 母公司公司章程副本
8.其他（如适用） 其他支持申请的文件
B. 企业继承文件（如适用）（购买现有企业）
1.企业执照/许可 购买的拟议企业执照或许可的副本
2.税务登记 购买的拟议企业税务登记证副本
3.财务报表 购买的拟议企业过去三年的财务报表副本
4.公司章程 购买的拟议企业公司章程副本
5.股权协议 购买的拟议企业股权协议副本，证明股东之间关系
6.所有雇员的税务报表 最近纳税年度为所有雇员（全职和兼职）支付薪金的T4报表副本
7.其他（如适用） 其他支持申请的文件
C. 关键员工文件（每一个关键员工申请人）
1.关键员工申请表 每一关键员工申请人的关键员工申请表（完成并签署）副本
2.照片 上传每一关键员工申请人的照片副本，需符合以下条件：照片的尺寸大小必须至少35mm×45mm(1 3/8” x 1 3/4”），300dpi或413×531像素或更大，显示头部的完整正面图，面部位于照片中间，包括肩部顶部；头部从下颌到头顶必须在31mm-36mm之间（1 1/4” 或1 7/16”）；白底；用申请人的名字命名，如 “John Smith passport photo.jpg”。
3.护照 每一关键员工申请人完整的有效护照副本：护照每一页（包括所有空白页）的清晰副本，临时居住签证，入境印章和/或任何其他移民印章；如果护照有效期至少两年，需包括关键员工之前所有护照页副本。
4.关键员工家庭成员护照（如适用） 每一关键员工申请人随行家属的护照个人信息页副本，随行家属包括配偶或同居伴侣，22岁以下的子女及他们的孩子。
5.出生证明 每一关键员工申请人的出生证明副本，如不能提供原始出生证明文件，需提供可接受的国家文件副本（如适当政府机构提供的说明为何不能提供出生证明的文件）。
6.IRCC或CBSA出具的身份文件（如适用） IRCC或CBSA出具的当前及之前身份文件副本，文件包括工作许可、学习许可、临时居住签证和/或加拿大移民文件。
7.工作经验 证明每一个关键员工申请人在公司工作的工作经验的文件副本，包括：与公司签署的工作合同（过去60个月内至少36个月全职工作）；薪资证明，例如工资条、薪金记录、显示工资存款的银行报表或所得税文件。
8.简历 关键员工申请人的当前简历副本
9.其他（如适用） 其他支持申请文件
