QC魁省投资移民

最新消息： 魁省移民部2016年5月30日开始接受申请，截止日期是2017年2月28日。全球接收1,900个魁北克投资移民申请。其中，分配给中国的名额为1,330个。比 2015年增加了130个名额（法语水平在B2以上的申请人还是不受名额限制，可以随时递交申请）。名额由移民局发放给政府指定的18家在册基金公司，由基金公司分配。所有申请都递交到魁北克蒙特利尔移民局。

魁省商业移民包括以下3个类别：

  • 投资移民 Investor Program
  • 企业家移民 Entrepreneur Program
  • 自雇移民 Self-Employed Worker Program

本网站着重介绍： 魁省投资移民。

项目历史

  1. 魁北克投资移民从2012年4月起实行递交配额制。当时是4月12日之前递交的，资料齐全完整的，均被接收。
  2. 2013年9月递交的，为抽签制。 中国的申请人当初都是递交香港，中签的比率大概是3.2:1.
  3. 2014年魁省一拖再拖，没开放。
  4. 2015年1月，魁省再次开放1750个配额，依然给中国人1200个配额。 递交到加拿大蒙特利尔境内移民局。 这次实施的是基金配额制，即移民局把投资的配额分配到委任的基金公司，基金公司再根据移民代理手里客户的数量和质量来综合评定，给予申请的配额。 换句话说，只要能拿到基金的配额，申请人的申请成功率几乎是不成问题的。 这也导致了后面的局面超出了代理和基金的预期， 这1200个名额是否满目前还不得而知。 因为这次的要求魁省真的是对中国申请人动了真格的，针对中国申请人单独出具了材料清单。
  5. 魁北克投资移民2016年5月份再次开启，分配给中国的名额为1,330个。比 2015年增加了130个名额。

详细内容请参考飞出国论坛魁省投资移民项目介绍。

申请条件

  1. 年龄要求：年满18周岁。
  2. 资产要求：160万加币的净资产（合人民币800万左右）。净资产的构成可以是： 流动资产，房产，商业资产。净资产可以都是主申赚取的，也可以是部分赚取的，但是不能完全是配偶赚取的。
  3. 工作经验：近5年内至少有2年的管理经验。管理经验可以是在农业、商业、工业领域的企业或者在专门从事法律服务的企业获得，在国际机构、政府部门或者政府分支部门的高级管理经验也适用。管理经验的定义：可以是在一个合格企业或者管理5名以上全职雇员的财务、人事或者物料企划、管理、控制等职业的管理经验。
  4. 语言要求：无要求。
  5. 考察要求：无要求。
  6. 投资金额：投资人需要和有魁省投资项目授权经纪人或者受托公司（基金公司）签投资协议：一次性支付22万加币，不涉及返还；或者投资80万加币给魁省政府无偿使用5年，到期后返还。

详细内容请参考飞出国论坛魁省投资移民申请条件。

申请流程

  1. 申请人准备资料并递交
  2. 收到魁省档案号
  3. 补料（完善信息）
  4. 收到等面试通知
  5. 面试
  6. 面试合格，1个月投资，收到魁省移民局签发的甄选证书，资料递交到移民局审理；
  7. 收到移民局档案号
  8. 移民局对申请人进行安全监察并通知全家体检
  9. 全家取得加拿大签证

申请费用

魁省投资从13年8月调整申请费，不再按申请人数收费，而是按申请家庭收费；13年是1万加币；15年是10106加币。

处理周期

申请周期：1.5-2年之间。

  • 死亡证明，遗嘱或法庭判决书确认遗产的性质及资金和财产转移给继承人的证明文件 复印件
