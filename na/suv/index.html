加拿大联邦创业移民 Start-up Visa Program

飞出国：加拿大联邦创业移民项目（SUV）于2013年正式开启。该项目以具有技能和潜力在加拿大创立企业的企业家为目标，所创办的企业需符合以下条件：

  • 为创新企业；
  • 可以为加拿大人创造就业；
  • 可以在全球范围内竞争。

SUV项目将这些企业家与加拿大的指定机构联系起来，以帮助他们实现其创新项目。申请人需具备创新商业理念，并且能够得到一个或多个加拿大指定机构的支持，指定机构包括：

  • 风险投资基金：必须获得风险投资集团至少20万加币的投资；
  • 天使投资集团：必须获得天使投资集团至少75000加币的投资；
  • 企业孵化器：必须被接受参与企业孵化器的项目。（重点描述）

SUV申请条件

  1. 经营一家符合条件的企业，需满足以下条件：
    • 在得到指定机构的认可时：
      • 每位申请人持有该公司当时已发行股份的10%或以上股权（最多5人可作为所有者申请）；并且
      • 申请人及指定机构共同持有当时已发行股份的50%以上的股权。
    • 在获得永久居留权时：
      • 申请人在加拿大境内对该企业进行积极和持续的管理；
      • 企业运作的重要部分发生在加拿大；
      • 该企业是在加拿大注册成立的。
  2. 获得指定机构的支持信，申请人需要：
    • 联系指定机构，了解如何获得其支持；
    • 让机构相信申请人有一个值得支持的商业理念；
    • 获得指定机构的支持信。
  3. 满足语言要求：能够用英语、法语或两种语言进行交流和工作，将有助于申请人在加拿大的企业取得成功。申请人需参加指定机构的语言测试，并达到听说读写CLB 5的语言水平。
  4. 具备足够的安家资金：加拿大政府不为SUV新移民提供财政支持。递交申请之时，申请人需要证明具备足够的资金可支持全家在抵达加拿大之后的生活，不能向其他人借钱。需要的资金取决于家庭人数，这个金额每年都会有更新。

SUV申请流程（企业孵化器）

  1. 种子阶段和资格评估（第一个月）：
    • 收集所有必要信息，包括申请人的履历、语言水平、个人资产以及商业和工作经验相关的信息，以评估候选人是否具备参与SUV项目的资格；
    • 要求对创新创业理念进行描述，在这一阶段，该理念可能仍处于初步阶段，项目方团队将审核该理念，提出建议和修改，并在考虑加拿大环境的前提下对它进行重新设计。
  2. 初步的商业计划与公司创建（第二个月）：
    • 确认并发展企业家的经营理念，在这一阶段，对项目的创新、营销、生产、创造就业和财务方面有了更清晰的认识，之后将递交商业计划的初步版本进行审批；
    • 根据《加拿大商业公司法》成立一个联邦公司，一位加拿大律师将被任命为这家新成立公司的董事会成员 -- 这不仅符合加拿大法律，即25%的董事会成员由加拿大居民构成，而且还为公司提供指导、行政和管理支持，并为当地合伙人和企业孵化器提供信心。
  3. 完成商业计划，将项目提交给企业孵化器，并收到支持信（第四个月）：
    • 在这一阶段，申请人的创业理念将得到充分的发展，并将定制一份全面的商业计划；
    • 商业计划提交给企业孵化器审批，提交后，企业孵化器可能会需要更多的信息，或可能建议修改，以及可能希望与企业家沟通，以讨论该计划，项目方将确保申请人为这些讨论做好准备；
    • 一旦商业计划被接受，企业孵化器会给每一位申请人发一封支持信。
  4. 准备并递交永久居留申请（第五个月）：
    • 项目方将为申请人的永久居留申请做准备，在此阶段，需要填写申请表并收集证明资料，包括：
      • 语言考试成绩；
      • 无犯罪记录；
      • 政府收费；
    • IRCC可能会要求申请人及其随行家庭成员提供生物特征信息，在签发永久居留签证或确认永久居留之前，还必须进行体检。
  5. 创业和孵化的启动（第14到20个月）：
    • 申请人必须证明已经做出了特定商业计划的努力，并采取了具体的行动来建立企业；此时将执行商业计划的初始步骤，公司将开设银行账户，开展营销和品牌活动，开发公司网站。在某些情况下，企业将雇佣员工，发展伙伴关系，获得战略资源，并向业务发展迈出大步（这与第四阶段同步进行）；
    • 在这一阶段，创业投资是必要的，投资金额将因创业项目不同而各异，申请人至少需要投资10万加币，以确保公司在启动阶段有足够的流动资金。
  6. 永久居留：
    • 在对申请人的申请进行积极评估后，IRCC将给予企业家及其随行家庭成员永久居民身份；
    • 由于申请人现在已经成为永久居民，被任命的董事将辞去创业公司董事会的职务，申请人将继续管理他在加拿大的企业，并成为其增长和成功的主要驱动力，包括确保该企业拥有足够的资本，有可能需要申请人提供更多的投资。

SUV申请费及审理周期

  • 申请费：$2075起
  • 审理周期：12-16个月
